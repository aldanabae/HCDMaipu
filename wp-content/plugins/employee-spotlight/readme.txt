Employee Spotlight is a fully responsive plugin to showcase and manage your team members, staff in a beautiful, and powerful way.

== Description ==

Employee Spotlight displays employees (staff), team members, founders or just a single person in a 4 column circle grid page. This page may be created by easy set up assistant on plugin activation. 

* Employee Spotlight comes with two responsive views which adjusts to the device screen size. 
* Each team member(employee) information has its own page, including major personal social networks and contact links as icons. 
* Team Members(Employees) can be displayed in summary view as well. 
* Comes with two sidebar widgets which display featured and recent team members(employees) selected in the editor.
* Easily set your slugs for team members(employees), locations, tags and groups from settings page. Change employees to team, founders or anything you like.
* Custom fields are supported and displayed selectively from settings page for your team.
* Additional stack views for archives, groups, employee tags and locations.
* NEW Added ability to enable/disable any field, taxonomy from backend and/or frontend.
* NEW Added ability to set the page templates for single, archive and every taxonomy related to Employees (left sidebar, right sidebar and full width) without modifying the theme files through plugin settings page
* See the [screenshots](https://wordpress.org/plugins/employee-spotlight/screenshots/) for more details.
* Configured to work with [eMD vCard Extension](https://emdplugins.com/plugins/emd-vcard-extension/) to allow generation of vCards based on team member(employee) profiles
* Configured to work with [eMD CSV Import Export Extension](https://emdplugins.com/plugins/emd-csv-import-export-extension/) for bulk import/export from/to CSV files
* Configured to work with [eMD Active Directory/LDAP Extension](https://emdplugins.com/plugins/emd-active-directory-ldap-extension/) Import/update automatically all employee fields including photos (jpegPhoto and thumbnailPhoto) from Active Directory, OpenLDAP and other LDAP.

== Installation ==

The simplest way to install is to click on 'Plugins' then 'Add' and type 'Employee Spotlight' in the search field.

= Manual Installation Type 1 =

* Login to your website and go to the Plugins section of your admin panel.
* Click the Add New button.
* Under Install Plugins, click the Upload link.
* Select the plugin zip file from your computer then click the Install Now button.
* You should see a message stating that the plugin was installed successfully.
* Click the Activate Plugin link.

= Manual Installation Type 2 =

* You should have access to the server where WordPress is installed. If you don't, see your system administrator.
* Copy the plugin zip file up to your server and unzip it somewhere on the file system.
* Copy the "employee-spotlight" folder into the /wp-content/plugins directory of your WordPress installation.
* Login to your website and go to the Plugins section of your admin panel.
* Look for "Employee Spotlight" and click Activate.
